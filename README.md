# 🌿 Дневник чувств

Telegram-бот — мягкая практика внимания к себе. Не трекер настроения ради
графиков, а помощник, который несколько раз в день спрашивает, что ты
чувствуешь, помогает назвать эмоцию и заметить, с чем она связана. Вечером
присылает итог дня.

## Как работает

- `/start` — выбираешь, как часто и в какое время бот пишет.
- В выбранное время приходит вопрос: какое чувство → насколько сильно
  (а для «не знаю» — какого оттенка) → с чем связано (короткой фразой).
- После ответа переписка сворачивается в одну компактную карточку записи;
  неотвеченный вопрос гасится, когда приходит новый (актуален только последний).
- `/emotions` — короткий словарик чувств с пояснениями, доступен в любой момент.
- В конце дня — дайджест: топ чувств и хронология записей.

## Стек

aiogram 3.x · APScheduler · SQLite (aiosqlite) · python-dotenv

## Запуск

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env      # вписать BOT_TOKEN от @BotFather
python main.py
```

Бот работает через long polling. Параметры — в `.env` (`BOT_TOKEN`, опционально
`DB_PATH`, `DIGEST_HOUR`).
